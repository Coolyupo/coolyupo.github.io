simple tmux guide

新增Pannel

  1. ctrl + b shift + "

切換Pannel

  1. ctrl + b pgup or ctrl + b pgdn

刪除當前Pannel

  1. ctrl + b x

設定Pannel同步

  1. ctrl + b shift + :
  2. set synchronize-panes on

取消Pannel同步

  1. ctrl + b shift + :
  2. set synchronize-panes off

適當地調整Panel畫面

有5個常用的預設指令可以使用，首先使用tmux產出5個Pannel，接著使用下列指令。

  1. ctrl + b alt + 1

  1. ctrl + b alt + 2

  1. ctrl + b alt + 3

  1. ctrl + b alt + 4

  1. ctrl + b alt + 5 (平均分割)
