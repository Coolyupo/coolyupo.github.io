CKA 考試紀錄

前言

起源於12月中旬左右的某天跟前公司主管聊天聽到一些抱怨XD，被其他部門的人質疑明明是XX組卻沒有人有CKA證照，其實當下我的想法是能夠解決問題遠比持有這張證照重要許多。

況且我也不認為對於使用Kubernetes有一小段時間的我們，這張證照真的很難考嗎?好啊，不然我就準備個一周來考看看(原本是想說裸考，但真的還是會抖XD)，儘管我已經不在公司了，不過也因為現在是離職待業(耍廢)狀態，做任何事情都很隨興的狀態。

正文

準備

這次大概使用了一週準備考試，準備的方向也跟其他網友差不多。

  1. 前三天使用udemy上的cka教學(課程因為很便宜兩年前就買了但一直放置，講師也很有心一直在更新的樣子)，不過我一堂課都沒聽(聽了會來不及XD)，而是直接使用附贈的kodecloud test(udemy課程中會提供enroll code)，因為本身已經存在一些些經驗了，一路做下來倒也沒遇到太多問題，不過還是有些不熟的地方會需要想一下、查找文件或偷看答案XD
  2. 第四天就用報名CKA送的兩次Killer.sh模擬考的其中一次，題目是真的如傳聞的長又複雜，全部寫完大概也是花了4個小時多一點，總共25題+2個ExtraProblem+3個PreviewProblem，好在寫完後正確率還算蠻高的(11X/12X)不然應該真的會崩潰XD，寫完後就把答案快速都看過一遍確定一下問題，跟建議的解法(有助於提升速度的)，然後這天的下午我就直接預約了後天下午的考試(需要至少24小時前預約)
  3. 第五天，我就啟用第二次的Killer.sh，不過這次的問題內容跟第四天的竟然完全一樣......不曉得是不是因為我第一天的session還沒過期導致的，還是原本就會一樣。(正常啟用一個session後36h才會過期)雖然是都一樣但全部寫完也是花了我兩個小時剛剛好(orz
  4. 第六天，下午就開始考試啦，理論上可以提前15分鐘從portal按下take exam，不過實際上還是要看你的live monitor是不是ready了(不過如果超過15分鐘沒開始考試將會失去資格而且不能重考唷)，我題目總共17題，難度的水平本身是覺得還好，應該比較接近kodecloud的mock exam，整體做完我大概還剩40分鐘就回頭去檢查(還真的檢查到一題不小心寫錯的)
  5. 第七天，考完後24小時內會收到成績與是否通過，我大概也是在22個小時收到通過的通知。

迷思

  1. 因為自己是第一次考這種live monitor的考試，原本以為是跟監考官互相視訊，原來只是我自己分享視訊+桌面，然後跟監考官用live chat的方式交談(原本一直很害怕自己英聽不好，倒是鬆了一口氣)
  2. 因為害怕有看不懂的題目，特地安裝了google translate的plugin，不過幾乎沒用到，因為題目並沒有這麼艱難的單字或語法就是了，況且介面上還提供轉換三種語言(英文、日文、簡體中文)
  3. 我遇到的監考官人很好也很客氣。

小技巧

  1. 雖然自己不是很喜歡，不過熟悉kubectl imperative command對於CKA的考試幫助非常大(可能大部分的人是用alias k=kubectl，因為我自己一直沒有這種習慣所以就沒這樣做)，在具備足夠基礎知識的情況下如何更快速作答會是關鍵。例如下面:
    • kubectl run $pod --image=$image --label="a=1,b=2" --dry-run=client -o yaml > ans.yml
    • kubectl create deployment $deploymentname --image=$image --dry-run=client -o yaml > ans.yml
    • kubectl create sa 、 kubectl create clusterrole.....、kubectl create rolebinding......
  2. terminal內已經有內建kubectl autocomplete，可以盡情使用tab
  3. 自己本身是認為考試的難度並不算太難，只要在看到題目的時候能知道他是在問什麼，然後透過官方文件查找應該都是有辦法的，重點還是在於自己對kubernetes的熟悉度有多少。
  4. 雖然有人推薦用網友分享的CKA書籤大全，但我自己實際上在用的時候更喜歡使用左邊的搜尋欄，根據我對題目的了解去查找我想要看到的資源文件(要注意不要點到會違規的連結就是了，我通常查出來的都是點第一個OR第二個XD)

題外話

這次考試有點緊張，因為選擇在家裡考，順帶把家人也弄得有點緊張XD，特地借用我姊的房間把它弄成家徒四壁的樣子，只剩一張桌子+一張子+延長線。(還被監考官誇獎準備的非常小心XD)。不過當下有點尷尬的是因為房間實在太空、又密閉竟然會有一點點回音(Orz)，然後我家附近住戶又有人在微施工不定時產生一些小噪音......，好加在不需要用麥克風對話(純打字)
